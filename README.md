# ISURS - Item Shortage & Unavailability Reporting System

## Dashboard
- 4 summary cards: Total Items, Shortages, Not Available, Total Reports
- Bar chart: reports per store
- Line chart: reports trend per week
- Recent reports table (10 latest)
- Export Excel & Print buttons

## Items / Master List
- **Add** item with Code, Name, Store
- **Edit / Delete** existing items
- **Import from Excel** (Master List2 sheet)

## New Report
- Form: select Item, Week, Unit, Ward (auto-loads per Unit), Section, Status
- Status: Shortage / Not Available

## Reports
- Naka-filter by Store, Status, Week
- Pagination (50 per page)
- Import from Excel (Report sheet)
- Export Excel (with current filters)
- Print

## Shortage Summary
- Items ranked by most reported shortage
- Filter by Store | Export Excel | Print

## Not Available Summary
- Items ranked by most reported unavailability
- Filter by Store | Export Excel | Print

## Settings
- **Stores** — add/delete
- **Units** — add
- **Wards** — add per Unit
- **Sections** — add
- **Weeks** — add/with date range
- **Seed from Excel** — import reference data
